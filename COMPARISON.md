## Key Differences

### AI Agent Integration
University Research MCP is built for AI agents using the MCP protocol. AI agents call tools like `research_university(name="MIT")` or `get_researcher_profile(name="John Smith")` directly — no manual database navigation, no complex query syntax.

ireseaerchnin is a free web tool for manual research. Elsevier is an enterprise platform requiring complex institutional access.

### Funding Data
University Research MCP uniquely includes NIH and NSF funding analysis alongside research output — combining publication data with grant data for the first time.

### Patent Landscapes for Universities
University Research MCP provides patent landscape analysis for university tech transfer evaluation — combining USPTO and EPO data with publication and funding data.

## When to Choose Each

**Choose University Research MCP when:**
- You're an AI agent doing tech transfer or VC research
- You need university + researcher + patent + funding in one call
- You want pay-per-use without institutional access
- You're building a tech transfer intelligence product

**Choose ireseaerchnin when:**
- You're doing manual academic research
- You don't need AI agent integration
- You're a researcher with free access

**Choose Elsevier when:**
- Your institution has existing access
- You need premium journal content
- You have budget for enterprise academic tools
